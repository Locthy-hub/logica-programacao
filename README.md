# 🧠 Lógica de Programação — Do Zero ao Avançado

> O curso gratuito mais completo de lógica de programação em português, feito para quem **nunca programou na vida** e quer sair pronto para aprender qualquer linguagem (Java, Python, C#, JavaScript, C++, Go, e outras).

Este repositório não é uma lista de exercícios. É um **curso estruturado**, com teoria explicada do zero, analogias do dia a dia, exemplos comentados, exercícios progressivos e projetos práticos.

---

## 📌 Para quem é este curso?

- Pessoas que nunca escreveram uma linha de código.
- Estudantes que já tentaram aprender e travaram na lógica.
- Quem está se preparando para faculdade, cursos técnicos ou processos seletivos de tecnologia.
- Autodidatas que querem uma trilha sólida antes de escolher uma linguagem.

Todo o conteúdo é escrito em **pseudocódigo / Portugol**, então não importa qual linguagem você vai aprender depois — a lógica é a mesma.

## 🗺️ Como usar este repositório

1. Comece pelo [Módulo 1](modulos/01-o-que-e-programacao.md) e siga a ordem — os módulos são cumulativos.
2. Depois de cada módulo, resolva os exercícios propostos antes de olhar o gabarito.
3. Use o [Guia de Estudos](GUIA-DE-ESTUDOS.md) e o [Cronograma](CRONOGRAMA.md) para se organizar.
4. Quando terminar um módulo, tente resolver alguns exercícios do [banco de exercícios](exercicios/) no mesmo nível.
5. Depois do Módulo 12 (Modularização), comece a praticar com os [projetos](projetos/).
